# 🧠 Turing Machine Simulator (Mtprak)

Проект представляет собой реализацию **машины Тьюринга** на языке **C++** с графическим интерфейсом, разработанным с использованием **Qt**.

Программа создана в учебных целях для демонстрации принципов работы машины Тьюринга и моделирования выполнения алгоритмов.

---

## 📌 О проекте

Машина Тьюринга — это абстрактная вычислительная модель, предложенная :contentReference[oaicite:0]{index=0}, которая используется для изучения теории алгоритмов и вычислимости.

Она состоит из:

- **ленты**, разделённой на ячейки
- **головки чтения/записи**
- **набора состояний**
- **таблицы правил переходов**

На каждом шаге машина:

1. считывает текущий символ с ленты
2. проверяет текущее состояние
3. ищет соответствующее правило
4. заменяет символ
5. сдвигает головку влево или вправо
6. переходит в новое состояние

Работа продолжается до тех пор, пока не будет достигнуто конечное состояние или не закончатся допустимые правила.

---

## ⚙️ Алгоритм работы

Общий алгоритм работы программы можно описать следующим образом:

1. Пользователь вводит **алфавит**
2. Задаются **правила переходов**
3. Вводится **начальная лента**
4. Машина начинает пошаговое выполнение
5. На каждом шаге:
   - читается символ
   - применяется правило
   - обновляется состояние
   - перемещается головка
6. Выполнение завершается при достижении финального состояния
